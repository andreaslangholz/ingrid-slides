04 · The Ingrid Gradient

The signature expression.

The gradient is a real PNG image, not a CSS gradient. Reserve it for key communications — hero visuals, campaigns, covers — and let it interact with the grid for depth rather than sitting as a flat overlay.

Violet–blue in the upper half, warm orange–gold below, violet dominating the center. The 2px progress bar is the only place a CSS approximation is used: linear-gradient(135deg,#6CA6F3,#AE76F2,#D981B7).

Application techniques
  • Mask into shape — apply the gradient to any form or container.
  • Soften & blend — feather edges; adjust with Freeform Gradient for seamless flow.
  • Morph organically — let it evolve into soft shapes that interact with key visuals.
  • Keep everyday usage minimal and intentional.
  • Do not recreate it with CSS radial-gradients — use the PNG.

07 · Visual System

Fluid Intelligence.

The system unites structure — the 2× Modular Grid — with expression — the Ingrid Gradient. The grid is the foundation for all brand layouts and typography, including every deck template.

Structure:
2× Modular Grid

Divisions of two

Built on divisions of two (2, 4, 8, 16, 32, 64 columns), it creates a flexible, logical framework for organizing space and content — grounded intelligence.

A graphic device

By exposing the underlying structure, the grid makes layout decisions intentional and transparent — reinforcing consistency and clarity.

Expression

The Ingrid Gradient is the organic counterpart — masked into the grid to add depth and emotional warmth. See gradient →

Constructing the grid
1 · Start at 2Divide the layout into 2 columns as the base grid.
2 · Scale by doublingExtend to 4, 8, 16, 32, or 64 columns as the layout demands.
3 · Modularize with intentionStyle into a modular framework — margins, gutters, baselines.
Usage example

A guide, not a cage — let key visuals and typography interact with the grid when it enhances expression. Maintain rhythm and balance through spacing and negative space.
